Project is created with:
* kotlin
* Mvvm Architcture pattern
* kotlin coroutines
* retrofit for networking
* ViewBinding
* stateFlow
* Repository pattern
* Clean arch
* Navigation Component
* Manual Dependency injection
* Junit4
